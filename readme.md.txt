Instrucciones de uso (ponelas en README.md)
Subí a GitHub: Crea repo, subí los 3 archivos, activá GitHub Pages.

Celular (Android/iOS Chrome):

tuusuario.github.io/turepo

Clic "MODO CELULAR" → "Iniciar stream" → copia ID.

PC (Windows Chrome):

Misma URL

Clic "MODO PC" → pega ID → "Conectar a celular".

Listo: PC ve video/audio del celular en tiempo real. Bidireccional si querés.

Funciona 100% remoto (PeerJS cloud maneja NAT/STUN). Para audio sistema en PC viewer, no hace falta Stereo Mix (solo recibe).
